1. Synchronization should be used for this assignment, since there is a producer-consumer problem involved. There are multiple threads trying to access the same memory area (the platform queue). We have planes trying to put products (passengers) to the platform, and meanwhile, taxis are taking them away from the platform. There is an obvious racing condition among the producers and consumers. First of all, if more than one producers are trying to put a passenger onto the platform, which passenger arrives first? This is crucial to the later decisions since the first one entered the platform must get out first. Secondly, when the platform is almost full, if more than one threads arrive at the critical section together, there is no way to determine which passenger gets onto the platform and which passenger take the bus. Therefore, synchronization needs to be used here to avoid these kind of production racing. Likewise, for taxis, there is no way to determine which taxi takes the first passenger.
